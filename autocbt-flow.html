AutoCBT · 多 Agent 序列图

展示 Agent 间消息传递与协作流程

第一轮对话
用户 → Counsellor: "我最近总是失眠，工作压力很大"
[内部] Counsellor 生成草稿，评估信心 = 0.6
路由决策: UNICAST → supervisor_identification
Counsellor → Identification: "评估认知扭曲"
[反馈] Identification: 检测到 catastrophizing + sleep-related anxiety
Counsellor ← Identification: 改进建议 → 更新草稿
第二轮路由决策
信心评估: 0.6 → 仍 < 0.8，继续路由
路由决策: MULTICAST → [empathy, strategy]
Counsellor → Empathy: "评估共情水平"
Counsellor → Strategy: "评估策略适当性"
Counsellor ← 整合建议: 改进回复草稿
第三轮 — 结束
信心评估: 0.85 ≥ 0.8 → ENDCAST
Counsellor → Final Reply: "整合监督者建议，生成最终回复"
Final Reply → 用户: "你的失眠可能与工作压力有关..."
安全机制
迭代计数: iteration < maxIterations (5)
循环检测: visitedSupervisors 无重复 → 正常
异常路径: 若检测到循环或达到最大迭代 → 强制 ENDCAST
