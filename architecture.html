模块详细说明

🧠 诊断系统 (Diagnosis Graph)

  • 精神科 8 步问诊流程（主诉→风险评估）
  • 心理科 8 步评估流程（认知/情绪/行为/人格）
  • 4 种专业化服务 Agent（儿童/青少年/正念/社交）
  • 自动症状检测 → 量表推荐 → 结果反馈闭环
  • 支持 PsyLLM 心理科专用模型

🔄 AutoCBT 多 Agent 系统

  • Counsellor Agent (a₀) 生成回复草稿并评估信心
  • 5 个 Supervisor 分别监督 CBT 五大原则
  • 5 种路由策略：LOOPBACK/UNICAST/MULTICAST/BROADCAST/ENDCAST
  • 最大迭代 5 次，信心≥0.8 自动结束
  • 基于论文 AutoCBT (arXiv:2501.09426)

📋 量表系统 (22 Scales)

  • 抑郁筛查：PHQ-9, SDS, BDI, HAMD, GDS, Burns
  • 焦虑评估：GAD-7, HAMA, STAI
  • 专项评估：ISI失眠, CDC认知扭曲, MoCA/MMSE认知
  • 专项人群：CERS/SDQ儿童, PSCI/YAPA青少年
  • 正念/社交：MAAS/FFMQ, SIAS/SAD

💬 Faye 情绪调节系统

  • 5 步流程：Identify → Explore → Reframe → Regulate → Reflect
  • 危机关键词检测（中英文）
  • 按情绪推荐循证练习（呼吸/接地/放松等）
  • CBT + 正念理论框架

🤖 LLM 基础设施

  • 主模型：agnes-2.0-flash via Agnes AI API
  • Headroom 压缩代理（token 节省 ~50%）
  • 本地截断降级（保留最近 6 条消息）
  • 模型后备链 + 限流指数退避
  • PsyLLM 适配器（心理科专用，支持 thinking chain）

📡 API 路由一览

  • POST /api/chat — 流式对话
  • POST /api/assess — 量表分析
  • POST /api/comprehensive — 多量表综合
  • POST /api/diagnosis/*/graph — LangGraph 诊断
  • POST /api/autocbt/chat — AutoCBT 对话
  • POST /api/faye — Faye 情绪调节
